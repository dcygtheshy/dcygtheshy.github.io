# dcygtheshy.github.io

A bilingual personal blog for 蛋蛋鼠, built with [Astro](https://astro.build/) and customized from [Fuwari](https://github.com/saicaca/fuwari).

## Local Development

```powershell
D:\codex\tools\node-v22.12.0-win-x64\corepack.cmd pnpm install
D:\codex\tools\node-v22.12.0-win-x64\corepack.cmd pnpm dev
```

## Build

```powershell
D:\codex\tools\node-v22.12.0-win-x64\corepack.cmd pnpm build
```

## GitHub Pages

This repository is prepared for a user Pages site:

- Repository name: `dcygtheshy.github.io`
- Site URL: `https://dcygtheshy.github.io`
- No Astro `base` path is needed for a user Pages repository.

Before publishing, replace the demo avatar and banner images if needed.

The GitHub Actions workflow builds the site with pnpm and deploys `dist/` to GitHub Pages.
